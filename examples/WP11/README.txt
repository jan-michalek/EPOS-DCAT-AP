Please, add in this folder your XML example files (EPOS-DCAT-AP_WPXX.xml) according to the generic example available on
https://github.com/epos-eu/EPOS-DCAT-AP/tree/master/examples/EPOS-DCAT-AP_example.xml

Please add DDSS-ID into the filename. For some of the webservices it is difficult to find out. If there are more webservices serving several DDSS elements in one file please assign DDSS-IDs in a comment for each WebService entity. 
The latest version of DDSS Master Table is here:
https://owncloud.epos-ip.org/index.php/apps/files?dir=/Shared%20Folder/8%20TRANSVERSAL%20WPs/WP06%20%26%20WP07/DDSS-Master-Table
